Page de présentation et de commande de produit de la boutique en ligne Kboutik 

-->  Présenter les produits  (photo/video , nom , description , nombre en stock , lieu de vente officiel , prix) 
--> Création de commande  (nomClient , numéro , lieu de livraison , quantité , sp produit ) - après envoie de la commande un message est envoyer sur le numéro du client pour confirmer la réception de sa commande 

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
